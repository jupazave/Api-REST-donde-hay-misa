API REST Donde hay misa Alpha

Servicio basado en el desarrollo de ajamaica utilizando los datos de Donde hay misa para la creacion de una aplicacion movil.

Par obtener datos:

  • Lunes: /lunes/{id_estado}
  • Martes: /martes/{id_estado}
  • Miercoles: /miercoles/{id_estado}
  • Jueves: /jueves/{id_estado}
  • Viernes: /viernes/{id_estado}
  • Sabado: /sabado/{id_estado}
  • Domingo: /domingo/{id_estado}
